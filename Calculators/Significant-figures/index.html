Significant Figures Calculator

How to count Sig Figs:

  1. Determine if there is a decimal.
  2. If a decimal is present, count from the left side starting from the first nonzero number. For instance, 0.00320 has three sig figs.
  3. If a decimal is not present, count from the right side starting from the first nonzero number. For instance, 9000 has one sig fig.
  4. If the number is extremely large, you can us this program.

Output:

0 sig figs
